ABOUT MALAY COMPLEX WORD IDENTIFICATION SYSTEM

Greetings and welcome to Malay Complex Word Identification System. This system was established back in November of 2021 by Phitsanuk Chiranont under the supervision of Ts. Dr. Juhaida Abu Bakar.

This system intended to be use as a learning tool for anyone who are interested or in the process of learning Malay language. It will help users to identify the prefix and affix words in Malay language.

User Manual

Image Grid

Click on the buttons to change the grid view.



STEP 1: Access to our website via the given link. User will be redirect to this greeting page. There are three tab can be interact with. Home tab represent greeting page, user manual tab and about us tab. User must click on Login/Register section to login into the system.





Greeting Page





STEP 2: Once user click on Login/Register, user will be redirect to this dashboard page. User can choose either to register new account, login or return to greeting page.





STEP 3: In order to use the system, user must register into the system. User must give the valid email format and password must follow the pattern (at least one uppercase, one lower case, one number and password must not less than eight characters).





STEP 4: After successfully registered, user must login with the registered email and password.



STEP 5: After successfully login user will be redirect to this page.



STEP 6: Copy text from any source.





STEP 7: Paste it inside the text box.





STEP 8: Select task. All for detecting all complex words inside the text. Noun, verb and adjective for detecting complex words associated with noun, verb and adjective respectively. Altered root words for detecting the complex words that slightly change from its root word. Press submit button to submit the text.





STEP 9: The results will be display in result field.
